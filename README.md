## 🔎 Project Overview

This project analyzes a real-world sales dataset using **SQL Server** to extract meaningful business insights.

The goal was to simulate real business scenarios and answer key revenue and performance questions using structured queries.

---

## 📁 Dataset

- **Source:** Kaggle – Sample Sales Data  
- Contains:
  - Orders
  - Customers
  - Products
  - Regions
  - Sales Revenue
  - Order Dates

---

## 🎯 Business Questions Solved

### 1️⃣ Top 5 Product Lines by Revenue
Identified highest revenue-generating product categories.

### 2️⃣ Monthly Sales Analysis
Analyzed revenue trends by month and year.

### 3️⃣ Region-wise Estimated Profit
Calculated estimated profit by territory.

### 4️⃣ Customer Lifetime Value (CLV)
Identified high-value customers based on total purchases.

---

## 🛠 Tools & Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- GitHub

---

## 🧠 SQL Concepts Applied

- `GROUP BY`
- Aggregate Functions (`SUM`)
- `ORDER BY`
- `TOP`
- CTE (Common Table Expressions)
- Window Functions (`LAG`)
- Basic Profit Calculations

## 🚀 Key Insights

- Classic Cars generated the highest total revenue.
- Sales showed seasonal variation across months.
- Certain territories consistently outperformed others.
- A small number of customers contributed significantly to total revenue.

---

## 📌 How to Run This Project

1. Import dataset into SQL Server.
2. Open `business_analysis.sql`.
3. Execute queries inside `sales_analysis` database.
4. Review results in SSMS.
